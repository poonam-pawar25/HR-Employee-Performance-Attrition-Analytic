# HR-Employee-Performance-Attrition-Analytic
HR analytics project using SQL, Excel, and Power BI to analyze employee performance and attrition trends.
#Project Overview
This project focuses on analyzing employee performance and attrition patterns to help HR teams make data-driven decisions.  
The analysis identifies key factors influencing employee turnover and performance using SQL, Excel, and Power BI



#Dataset Description
The project uses 4 related HR tables

#Employees Table
- Employee_ID  
- Age  
- Gender  
- Department  
- Job_Role  
- Education  
- Hire_Date  

 #Performance Table
- Employee_ID  
- Performance_Rating  
- Training_Hours  
- Last_Promotion_Year  

#Compensation Table
- Employee_ID  
- Monthly_Income  
- Salary_Hike_Percentage  
- Stock_Option_Level  

#Attrition Table
- Employee_ID  
- Attrition 
- Years_at_Company  
- Work_Life_Balance  



#Tools & Technologies
- SQL– Data extraction, joins, aggregations  
- Excel – Data cleaning, validation, preprocessing  
- Power BI – Data modeling, DAX, dashboards  



#Data Analysis Workflow

#SQL – Data Extraction
- Joined 4 tables using Employee_ID
- Calculated:
  - Attrition rate by department
  - Average performance rating
  - Salary vs attrition analysis
- Used: JOIN, GROUP BY, CASE, COUNT, AVG

#Excel – Data Cleaning
- Removed duplicates  
- Handled missing values  
- Standardized categorical columns  
- Validated numeric fields  

#Power BI – Visualization
 Created data model and relationships  
- Developed DAX measures:
  - Attrition Rate  
  - Average Performance Score  
- Built interactive HR dashboards  



#Key Dashboards
- Overall Attrition Rate  
- Department-wise Attrition  
- Performance vs Attrition  
- Salary Impact on Attrition  
- Work-Life Balance Analysis  



#Key Insights
- Employees with lower performance ratings show higher attrition  
- Lower salary bands have increased employee turnover  
- Certain departments experience consistently higher attrition  
- Work-life balance plays a significant role in employee retention
